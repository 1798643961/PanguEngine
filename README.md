基于Java的体素沙盒游戏引擎。**等你为它赋值。**

A voxel sandbox game engine based on Java. **Waiting for your assignment.**

## 安装（Install）
```
$ git clone https://github.com/UnknownDomainGame/NullEngine.git
$ cd NullEngine
$ gradlew install
```
